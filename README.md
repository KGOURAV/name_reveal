# Baby Reveal Website

This project is a simple and modern website designed to reveal the name of your baby along with a few photos. The website is structured to be easily hosted on GitHub Pages.

## Project Structure

```
baby-reveal-site
├── docs
│   ├── index.html          # Main HTML document for the website
│   ├── assets
│   │   ├── css
│   │   │   └── styles.css  # Modern CSS styles for the website
│   │   └── js
│   │       └── script.js    # JavaScript for interactive features
├── images                  # Directory for baby images
├── .gitignore              # Specifies files to ignore in Git
└── README.md               # Project documentation
```

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```
   git clone https://github.com/yourusername/baby-reveal-site.git
   ```

2. **Navigate to the project directory**:
   ```
   cd baby-reveal-site
   ```

3. **Open the `docs/index.html` file in your web browser** to view the website.

## Customization

- **Baby Name**: Update the baby name in the `index.html` file.
- **Photos**: Add your baby photos to the `images` directory and update the image references in `index.html`.
- **Styles**: Modify the `styles.css` file to change the appearance of the website.

## Deployment

To deploy the website on GitHub Pages:

1. Push your changes to the `main` branch.
2. Go to your repository settings on GitHub.
3. Scroll down to the "GitHub Pages" section and select the `main` branch as the source.

Your website will be available at `https://yourusername.github.io/baby-reveal-site`.
